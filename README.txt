YouTube Downloader - User Guide
==============================

This application allows you to download videos from YouTube in various formats.

Installation Instructions:
-------------------------
1. Extract all files from the ZIP archive to a folder on your computer
2. Right-click on "install-and-run.bat" and select "Run as administrator"
3. Wait for the installation to complete (this may take several minutes)
4. Once installation is complete, you can start the application

Starting the Application:
------------------------
1. Double-click on "start-app.bat"
2. The application will open in your default web browser
3. Enter a YouTube URL and follow the on-screen instructions to download videos
4. To stop the application, press any key in the command window

Creating a Desktop Shortcut (Optional):
--------------------------------------
1. Run "create-shortcut.bat" to create a desktop shortcut
2. You can then start the application directly from your desktop

Emergency Shutdown:
-----------------
If the application is not responding or you need to force it to close:
1. Run "force-shutdown.bat" to immediately terminate all application processes

Troubleshooting:
---------------
- If the application doesn't start, make sure you've completed the installation
- If you see any error messages, try running the install-and-run.bat file again
- If problems persist, contact the person who provided you with this application

Note: This application requires an internet connection to download videos.
